1. Project Overview

Nano Bijoux is a modern, multilingual e-commerce website built for an Algerian jewelry and accessories boutique (@nano31bijoux). The goal is to create a complete online store that allows customers across all 58 Algerian wilayas to browse products, add to cart, and place orders with local payment methods (Cash on Delivery, BaridiMob, CCP).

2. Technology Stack

Every technology was chosen with purpose — balancing modern DX, performance, and suitability for the Algerian market:

Technology Version Role Why Chosen
Next.js 16.2.1 Framework SSR + App Router for SEO, fast page loads on slower connections
React 19 UI Library Component-based architecture, huge ecosystem
TypeScript 5.x Type Safety Catch errors at build time, better IDE support
Tailwind CSS 4 Styling Rapid UI development, built-in RTL support for Arabic
Supabase Latest Backend Free tier, Auth + DB + Storage in one, PostgreSQL-based
Zustand Latest State Management Lightweight, localStorage persistence for cart/wishlist
next-intl Latest i18n Best Next.js i18n library, supports FR/AR/EN with RTL
Framer Motion Latest Animations Smooth, performant UI animations
React Hook Form + Zod Latest Forms Performant forms with schema-based validation

  • Supabase Database Setup — Create tables for products, orders, users, categories, reviews
  • Authentication — Supabase Auth for login/register, protected routes, session management
  • Real Product Data — Replace mock data with database queries
  • Order Processing — Store orders in database, order status tracking
  • Payment Integration — BaridiMob, CCP, Cash on Delivery confirmation
  • Email Notifications — Order confirmation, shipping updates
  • Search — Full-text product search
